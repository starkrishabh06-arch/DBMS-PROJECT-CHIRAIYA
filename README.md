# ExpenseAI 💸

A full-stack web application designed to help users efficiently manage, track, and analyze their daily expenses. This project provides a simple and intuitive interface for recording financial activities, categorizing expenses, and gaining insights into spending habits.

## ✨ Features :

* Authentication (Login / Signup)
* Dashboard
* Expense Module (Add / Manage)
* Income Module (Add / Manage)
* Lending Module (Add / Manage)
* Analytics Module (Graphs & Insights)
* Reporting Module
* Settings Module

## 🌐 Web Installation and Setup Guide

1.Launch the XAMPP or WAMP Control Panel and start the Apache and MySQL services.

2.Extract the downloaded project ZIP file to your system.

3.Move the extracted project folder:
       -XAMPP → htdocs directory
       -WAMP → www directory

4.Access phpMyAdmin via your browser:
           http://localhost/phpmyadmin

5.Create a new database and import the expenditure.sql file from the database folder.

6.Navigate to the project in your browser:
           http://localhost/DBMS-PROJECT-CHIRAIYA-main

7.Configure database connection settings in the .env file as needed.

## 🧠 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Server: Apache (XAMPP/WAMP)

## 🎯 Objective

The goal of this project is to demonstrate how database-driven applications can be used to manage personal financial data effectively, while providing meaningful insights into spending behavior.
